No debe ser misterio para cualquiera que venga a visitar esta página el hecho de que Github ha sido comprada por Microsoft. Sí, es verdad. En un lapso a futuro podría resultar en un montón de teorías locas acerca de como es que Microsoft destruirá un producto como este.

En mi opinión no solicitada, tengo que decir que no hay mucho que pueda agregar. Microsoft tiene la fama de dañar mucho los productos que adquiere.

La decisión personal que he tomado es que esta página seguirá en github hasta que ya no se pueda más, sin embargo mis repositorios y archivos de configuración serán almacenados dentro de Gitlab. En algún momento, si llego a tener el dinero suficiente, mudaré este sitio a un hosting propio. No se requiere de mucho, por lo cual no siento que sea tan terrible que siga por acá.

Si las cosas se ponen feas, les aviso. ;p
